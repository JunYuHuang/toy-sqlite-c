# Toy SQLite in C

This is a following of the tutorial for [building a SQLite clone from scratch in C](https://cstack.github.io/db_tutorial/).

## How To Test

Run the following in a Bash terminal:
```bash
# Run this
bundle exec rspec

# OR this
rspec
```
